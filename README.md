# Estudos de Terraform

Este repositório tem como objetivo **registrar meus estudos e experimentos com Terraform**.  
Não se trata de um projeto de produção, mas sim de um espaço para aprendizado e prática.

## Aviso Importante
- Os exemplos e configurações aqui presentes **provavelmente não seguem as melhores práticas**.  
- O foco é **explorar recursos e entender conceitos**, não criar infraestruturas otimizadas ou seguras para uso real.  

## Estrutura
- Cada diretório ou arquivo pode representar um teste ou exercício específico.  
- Alterações frequentes e commits podem refletir ajustes durante o processo de estudo.  

## Objetivo
- Consolidar conhecimento sobre Terraform.  
- Ter um histórico da evolução dos estudos.  
- Servir como referência pessoal para consultas futuras.

---

> ⚠️ **Nota:** Este repositório não deve ser usado como base para ambientes de produção.
